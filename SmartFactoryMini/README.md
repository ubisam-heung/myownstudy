# SmartFactoryMini

Demo smart-factory simulation (MES + MCS + SCADA + ECS + PLC) in a single console app.

Run (requires .NET 10 SDK):

```bash
dotnet run --project SmartFactoryMini/SmartFactoryMini.csproj
```

Or:

```bash
cd SmartFactoryMini
dotnet run
```

Notes:
- If you have a different SDK, change `TargetFramework` in the csproj accordingly.
